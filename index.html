Smart Saving For Eclipse 4

Smartsave intelligently saves your files after you stop editing in Eclipse4.

Features

  • Intelligently saves files when you stop editing.
  • Intelligently saves files when editor loses focus.
  • Option to change save interval.
  • Option to prevent saving if compiler warnings/errors are present.
  • Option to toggle smart saving.

Installation Guide

  1. Launch Eclipse
  2. Go to Help -> Install New Software...
  3. Press the Add button on the install window
  4. Paste the update site link to location.
  5. Press OK
  6. Select the Smartsave plugin shown in the list view under Uncategorized.
  7. Press Next.
  8. Agree to the license terms.
  9. Press finish.
  10. Restart Eclipse.
